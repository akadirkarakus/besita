# Embedded Systems (Gömülü Sistemler)

Bu klasör projenin gömülü sistem kısımlarında bulunacak olan kodları içerir.

## Genel Açıklama
Analog to Digital Converter (ADC) cihazından çıkan ağırlık verileri ESP32 mikrodenetleyicisine gelir.
Yüzlerce tartım kaydından oluşan bu veriler JSON formatında paketlenerek Raspberry Pi'ye gönderilir.

## ESP32 Görevleri

1. **Veri Toplama**
   - ADC portundan anlık olarak ağırlık sensörü verisini oku
   - Hata kontrolü (sensor bağlantısı, aralık kontrolü)

2. **Veri Paketleme**
   - Verileri JSON formatında düzenle
   - Örnek paket:

3. **İletişim**
   - Raspberry Pi'ye TCP/UDP veya MQTT ile bağlantı
   - Bağlantı durumu kontrolü
   - Yeniden bağlantı mekanizması

4. **Gönderme**
   - JSON paketlerini Raspberry Pi'ye gönder
   - Başarı/hata logları

## Teknolojiler

- ESP32: Ardunio IDE veya PlatformIO
- Sensör: Load cell (ağırlık sensörü) + ADS1232 ADC Module
- İletişim: WiFi (TCP/UDP) veya Bluetooth (opsiyonel)
- Veri Formatı: JSON
